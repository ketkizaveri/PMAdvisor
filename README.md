PMAdvisor
=========

PMAdvisor is a first-version product management assistant interface built with
Next.js, TypeScript, and Tailwind CSS.

Current scope:

- Home page with title, subtitle, search, and example questions
- Ask page with the submitted question, placeholder answer, and citation cards
- Reusable SearchBox, CitationCard, and QuestionExamples components

AI, vector database, and retrieval integrations are intentionally not included
yet.

Run locally:

```bash
npm install
npm run dev
```
